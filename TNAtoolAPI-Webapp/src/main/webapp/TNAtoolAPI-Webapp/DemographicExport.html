Transit Network Analysis Software Tool - Demographic Export



Export demographics information to CSV. This tool will select all demographics information in the database for the census blocks in the selected counties. Each row contains the basic details of a census block in the initial columns, and details of the associated census tract, census place, urban region, county, state, and congressional district. The header in each of the columns is prefixed with the source table, e.g. "census_tracts_population2050". Data includes census geographic IDs, land and water areas, the current population, projected future populations, and a central latitude & longitude for each area. The next set of columns is all associated Title VI data for each census block. The last set columns includes LODES data (RAC & WAC) for each block, followed by LODES RAC projections for the block and associated county.

Note: Excel may have difficulty opening large CSV files (e.g. > 100 MB). If this happens, please select fewer counties.

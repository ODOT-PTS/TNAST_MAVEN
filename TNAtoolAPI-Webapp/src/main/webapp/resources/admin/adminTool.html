Notes:


  • The only accepted format for Title VI data is CSV.
  • Six different files must be uploaded. The name and the format (column order and names) of the files must exactly be according to the sample files provided below.
    1. b03002.csv
    2. b16004.csv
    3. b17021.csv
    4. b18101.csv
    5. b19037.csv
  • These files can be downloaded from www.nhgis.org according to the following instructions:
    1. Go to www.nhgis.org
    2. Use the "Log in" button to log into your account. You can create a new account for free, if you do not have one.
    3. Once logged in, go to the "Select Data" (or "main") page.
    4. Click on "Geographic Level" button and select "Block Group" (except for b18101 table for which you should select "Census Tract").
    5. Click the "Submit" button.
    6. Click on "Years" button and select the most recent range from the list of "5-Year Ranges".
    7. Click the "Submit" button.
    8. In the list of tables provided at the bottom of the page, look for the table name you want to download (e.g. b03002).
    9. Once found, select that table by clicking on the "plus" sign on its left.
    10. Click the green "Continue" button on the top-right corner of the page.
    11. Click "Continue" again.
    12. On the "Review and Submit" page, under "Table File Structure", make sure the "Comma delimited (best for GIS)" is selected only.
    13. Under "Source Tables", click on the "Geographic Extents" button and select the desired state.
    14. Submit the request.
    15. It will take you to the download page in which you will see a table of your requests.
    16. You have to wait until the table is ready to download. It usually takes a few minutes.
    17. Once available, you can click on its name and download the file.
    18. Once available, you can click on its name and download the file.
    19. There will be two CSV and two TXT files (the txt files are the metadata files for the csv files).
    20. You are going to use the CSV file that ends with an "E" (which stands for estimates).
    21. The column names are in codes. You can use the metadata txt file to find the columns you need.

Notes:


  • The only accepted format for Employment data is CSV.
  • Two files must be uploaded. The name and the format (column order and names) of the files must be exactly according to the sample files provided below.
    1. or_rac_S000_JT00.csv
    2. or_wac_S000_JT00.csv
  • These files can be downloaded from lehd.ces.census.gov/data/ a follows:
    1. Once the page is loaded, scroll down to get to the "LEHD Origin-Destination Employment Statistics (LODES)" section.
    2. Select LODES7 from the "version" dropdown list.
    3. Select the desired state from the "state" dropdown list.
    4. Select RAC from the "type" dropdown list.
    5. Click on "view files" button.
    6. From the list, find the file called "or_rac_S000_JT00_[desired year].csv.gz".
    7. Right click on the file and save the file.
    8. Decompress the file to get the "or_rac_S000_JT00_[desired year].csv" file.
    9. Rename the file to "or_rac_S000_JT00.csv" file.
    10. Repeat steps 4 to 9 for WAC data type.

Notes:


  • Title VI data must be uploaded in a file using the Comma Separated Value (CSV) format. No other formats are accepted.
  • Five different CSV files must be uploaded at the same time (as shown in the list below). The name of the files, as well as the names and the order of the columns in the CSV files to be uploaded, must be exactly the same as those shown in the example CSV files accessible via their corresponding hyperlink.
    • File NameDescription
    1.b03002.csvHispanic or Latino Origin by Race
    2.b16004.csvAge by Language Spoken at Home by Ability to Speak English for the Population 5 Years and Over
    3.b17021.csvPoverty Status of Individuals in the Past 12 Months by Living Arrangement
    4.b18101.csvSex by Age by Disability Status
    5.b19037.csvAge of Householder by Household Income in the Past 12 Months (in 2015 Inflation-Adjusted Dollar
  • The data for each of the five files listed in the table above must be requested and downloaded separately from the website www.nhgis.org by following these steps:
    1. Go to www.nhgis.org.
    2. Click on the "Log in" button to log into your account. If you do not have an account, you can create one for free.
    3. Once you log in, the "NHGIS Data Finder" page will be displayed automatically which contains four buttons labeled "Geographic Levels", "Year", "Topics", and "Datasets".
    4. Click on the button labeled "Geographic Levels". This action will open a window that displays different levels for which Title VI data can be selected (e.g., Nation, State, County, etc). To select one of these levels, you must click on the corresponding yellow plus sign (+) icon.
    5. To access the data to create the files "b03002.csv", "b16004.csv", "b17021.csv", and "b19037.csv", the level "Block Group" must be selected by clicking on the corresponding yellow plus sign. To access the data to create the file "b18101.csv", the level "Census Tract" must be selected. Once the appropriate level has been selected, click the "Submit" button. REMINDER: Each of these files must be created separately.
    6. Click on the button labeled "Years" and then select the most recent range from the list labeled "5-Year Ranges." Click the "Submit" button when done.
    7. The table displayed in the bottom half of the browser window now lists tables constrained by the year range just selected. The second column of this table is labeled "Table Name". Look for the name of the table you want to download (e.g., "B03002"). You may need to use the page navigation controls shown at the bottom of the browser window.
    8. Once found, select the table by clicking on the yellow plus sign icon displayed on leftmost column.
    9. Scroll to the top of the browser window and click on the green button labeled "Continue" on the Data Cart panel.
    10. Click "Continue" again.
    11. On the "Review and Submit" page under "Table File Structure," make sure the option "Comma delimited (best for GIS)" is the only one selected.
    12. Under "Source Tables," click on the button labeled "Geographic Extents" and select the desired US state (e.g., Oregon). Click on the button labeled "Submit".
    13. On the "Review and Submit" page, click on the button labeled "Submit" displayed at the bottom of the page. This action will open the "Extracts History" page which shows all data requests.
    14. Wait until your table is ready to download. This process usually takes a few minutes.
    15. Once the data request has been processed (this usually takes a few minutes), the NHGIS site will send an automatic email to the address used for registration. Alternatively, the browser page "Extract History" can be refreshed until the column labeled "Download Table Data" displays a hyperlink.
    16. Click on the hyperlink labeled "tables" to download a ZIP file containing the Title VI data for the selected table (e.g., B03002).
    17. The column names of the CSV files are coded. You can use the metadata TXT file to identify the columns you need.
    18. Rename the CSV file according to the ones specified in the table above (e.g. b03002.csv or b16004.csv).

Notes:

Notes:


  • Employment data must be uploaded in a file using the Comma Separated Value (CSV) format. No other formats are accepted.
  • Two files with employment data must be uploaded at the same time.
  • The name of the files, as well as the names and the order of the columns in the CSV file to be uploaded, must be exactly the same as those shown in the following sample CSV files:
    1. rac.csv
    2. wac.csv
  • Employment data to create the required files can be downloaded from lehd.ces.census.gov/data/ by following these steps:
    1. Once the lehd.ces.census.gov/data/ page is loaded, scroll down to the section titled "LEHD Origin-Destination Employment Statistics (LODES)"..
    2. Select LODES7 from the dropdown list labeled "Version".
    3. Select the desired state from the dropdown list labeled "State".
    4. Select Residence Area Characteristics (RAC) from the dropdown list labeled "Type".
    5. Click on the button labeled "View Files".
    6. From the list, find the file called "[desired state]_rac_S000_JT00_[desired year].csv.gz".
    7. Right click on the hyperlinked file name and save the file to a desired location.
    8. Decompress the file to get the "[desired state]_rac_S000_JT00_[desired year].csv" file.
    9. Repeat steps 4 to 8 to download WAC data.
    10. Rename the files as "rac.csv" and "wac.csv".
